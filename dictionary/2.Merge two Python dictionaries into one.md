```
input:  
dict1 = {'Ten': 10, 'Twenty': 20, 'Thirty': 30}  
dict2 = {'Thirty': 30, 'Fourty': 40, 'Fifty': 50}  

output:  
{'Ten': 10, 'Twenty': 20, 'Thirty': 30, 'Fourty': 40, 'Fifty': 50}
```

```python
dict1 = {'Ten': 10, 'Twenty': 20, 'Thirty': 30}
dict2 = {'Thirty': 30, 'Fourty': 40, 'Fifty': 50}

    d[i]=dict1[i]
```

```python
    for key,value in y.items():
```
