```
Given two strings, s1 and s2. Write a program to create a new string s3 by appending s2 in the middle of s1.

Given:
s1 = "Ault"
s2 = "Kelly"

Expected Output:
AuKellylt
```

---


```python
s1 = "Ault"
s2 = "Kelly"

#[start:end:step]
a=s1[0:len(s1)//2]
b=s1[len(s1)//2:len(s1)]
c=""
c=c+a
c=c+s2
c=c+b

print(c)
```
