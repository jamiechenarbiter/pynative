```
input  
Input1: 1 1 2 3 4 5 1 2 1 
Output1: 2 3 4 5 2 
Explanation : The input list is [1, 1, 2, 3, 4, 5, 1, 2] and the item to be removed is 1. 
After removing the item, the output list is [2, 3, 4, 5, 2]
```

___


```python
def func(n):
    a=[]
    for i in n:
        if i!=1:
            a.append(i)
    return a
list1=[1,1,2,3,4,5,1,2,1]
res=func(list1)
print(res)
```
